Included or Excluded Test Cases

In this page you can visualize all test cases included and/or excluded permanently from the test suite. For example, besides the filters mentioned in the Test Selection Page, you can include or exclude some test cases permanently, independently from the filters results.  

  1. Go to the Test Cases tab and  Include Test Case Permanently or Exclude Test Case Permanently of the Generated Test Cases list in the Test Cases Viewer clicking on right button of the mouse in the test case. The same behavior can be executed in the Traceability Matrix page. The all test included/excluded permanently  appear in the Included or Excluded Test Cases tab.

       

 

  1. The all test included/excluded permanently  appear in the Included Test Cases or Excluded Test Cases areas.

       

 

  1. In this tab you can also exclude a test case or cancel a test case inclusion in the Included Test Cases area and include a test case or cancel a test case exclusion in the Excluded Test Cases area:
  • Clicking on right button of the mouse in a test case of the Included Test Cases area you can visualize the Exclude Test Case and Cancel Test Case Inclusion option.

            

  • Clicking on right button of the mouse in a test case of the Excluded Test Cases area you can visualize the Include Test Case and Cancel Test Case Exclusion option.
