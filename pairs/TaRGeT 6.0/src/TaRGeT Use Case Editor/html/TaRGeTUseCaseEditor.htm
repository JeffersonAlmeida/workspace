TaRGeT Use Case Editor

 

Use Case Editor is a feature from TaRGeT used to create and edit use cases.  Based on Use Case Document XML schema, its interface makes easier to manipulate the document.  In the next sections, you will learn how to do the following actions using our editor and what kind of errors can be identified.

 

Actions that can be done in the Use Case Editor:

 

-          Creating new Use Case

o        Add use case to an old use case document

o        Create new Use Case Document

o        Errors messages

-          Editing a Use Case

-          Adding Alternative Flows

-          Removing Alternative Flows

-          Adding Steps

-          Removing Steps

-          Editing Steps

-          Searching Words

-          Exporting Use Case Documents
