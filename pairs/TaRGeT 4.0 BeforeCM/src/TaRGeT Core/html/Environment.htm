Environment

Here, non-functional requirements related to the environment compatibility and external software dependencies are described.

Compatibility

Desktop Platform

TaRGeT application shall work on Windows XP. In other Windows versions (NT and 2000), TaRGeT may not work properly.

Software Dependencies

Microsoft Office

TaRGeT application works properly with Microsoft Office 2003 Professional installed, since the imported use case documents contains XML content that is only handled by that Microsoft Office version.

.NET Runtime Environment

TaRGeT application only works properly in machines with .NET Framework 2.0 Runtime Components installed. If the machine does not have it installed, a warning message is presented.

JVM

TaRGeT application only works properly in machines with JVM compatible with JRE Version 6.0 installed. If the machine does not have the JVM installed, a warning message is presented.
