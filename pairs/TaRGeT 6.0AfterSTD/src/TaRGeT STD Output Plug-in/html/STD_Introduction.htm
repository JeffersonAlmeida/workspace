TaRGeT for System Test Development Team

 

This tool is a product from TaRGeT product line and aims to support the STD team in test case design process. It helps them to generate automatically test cases from use cases specification, improving your productivity and reducing the time spent to do this task.

 

 

So, its input is a spreadsheet containing the use case description and it returns another spreadsheet in a specific format with some information of test cases. This output can be modified by the user, providing him/her to adjust or insert some information that can not be processed automatically from the input.
