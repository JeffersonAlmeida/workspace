Traceability Matrix

In this page you can visualize the test cases according to the Requirements x Test Cases relation and the Use Case x Test Cases relation. Moreover, you can filters select,  and include / exclude test cases before test suite generate.  

  1. Select Tools > On The Fly Generation  from the menu bar. You can also open On The Fly Generation editor  via shortcut 'CTRL+Y'.

     

  2. Choose some filter in the Test Selection Page.

   

 

  1. Go to the Traceability Matrix. In this tab you can visualize the test cases that chosen according with the filters of the Test Selection Page. This test cases are displayed in two areas according to Requirements x Test Cases relation and Use Cases x Test Cases relation.

     

       

  1. You can save the filter clicking on Save Current Filter Selection button. After, you can select it in the combo selection list and load it clicking on Load Former Filter Selection button. See figures below:

           

 

 

 

 

  1. When you select the filter before of load it, if there are New Test Cases related to actual filter selected in the tab Test Selection Page , it appears in Requirements x Test Cases list and Use Case x Test Cases list with green font. The Common Test Cases are the test cases common between actual filter  in the tab Test Selection Page and the last filter selected in the Traceability Matrix tab.

       

 

  1. If you check the Removed Test Cases, you can  visualize the removed test cases related to the  actual filter  in the Test Selection Page tab and the last filter selected in the Traceability Matrix tab. It appear with red font in the Requirements x Test Cases list and Use Case x Test Cases list.
       

 

  1. You can visualize the test cases in spreadsheet format. If click in  a test case, you can visualize the selected test case into Selected Test Cases area.

       

 

  1. You can Include Test Case Permanently or Exclude Test Case Permanently of the Requirements x Test Cases list and Use Case x Test Cases list clicking on right button of the mouse in a test case. When you include or exclude the test case permanently, the test included/excluded will appear in the Inclusion/Exclusion tab.

       

 

           If you want cancel the test case inclusion, you can click on right button of the mouse in the included test case and choose the Cancel Test Case Inclusion option.

            

           If you exclude a test case, it will disappear the Requirements x Test Cases list or Use Case x Test Cases list . In this case, you can cancel the exclusion in the Inclusion/Exclusion tab. 

 

  1. If you want generate the test suite, go to the Test Cases Viewer page and click on Save Test Suite button.  If no filters are selected in the Test Selection Page or none test case is excluded of Test Cases Viewer tab, then all possible test cases are generated.
