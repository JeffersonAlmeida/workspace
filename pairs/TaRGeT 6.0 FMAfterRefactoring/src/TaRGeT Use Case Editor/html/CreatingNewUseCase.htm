Creating a New Use Case

 

It is easy to create a new use case using Use Case Editor.  First of all, you have to create a new project or open one already existent in TaRGeT. Then, go to the menu Tools and choose the Use Case Editor option.

 

 

 

The Use Case Editor will opened with all fields empty:

 

 

Fill in at least the mandatory fields and save all modifications using the icon or the shortcut Ctrl+S.
