Traceability Matrix

The test suite generation also includes the automatic generation of the traceability matrices. The matrices are included in the output Microsoft Excel document as spreadsheets. There is one spreadsheet for each traceability matrix. TaRGeT provides the following traceability matrices:

  • Requirements x Use Cases - this matrix links each requirement to its related use cases. The matrix is placed in the RTM - Requirements tab.

       

 

  • Requirements x Test Cases - this matrix links each requirement to the test cases that cover it. The matrix is placed in the RTM - System Test tab.

       

 

  • Use Case x Test Cases - this matrix links each use case to the related test cases. By related test cases, we mean that at least one step from the use case is contained in the test cases. The matrix is placed in the RTM - Use Case tab.
